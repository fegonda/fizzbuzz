Hello, Welcome to FizzBuzzer

by Felix Gonda

This page demonstrate the functionality of FizzBuzzer, a function that takes a number N as a parameter and prints the numbers from 1 to N to the page. But for multiples of three print "Fizz" instead of the number and for the multiples of five print "Buzz". For numbers which are multiples of both three and five print "FizzBuzz".
